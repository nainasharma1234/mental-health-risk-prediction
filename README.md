# Mental Health Risk Prediction System

## Description
This project predicts the mental health risk level of students based on their symptoms using machine learning and NLP techniques.

## Technologies Used
- Python
- Scikit-learn
- NLP
- Pandas, NumPy

## Features
- Symptom-based input
- Risk level prediction (Low / Medium / High)
- Probability output for each class

## Outcome
Achieved good prediction accuracy and helped in understanding real-world ML workflows.
